# protonhax

Это Rust-реализация скрипта [protonhax](https://github.com/jcnils/protonhax) для запуска программ в среде Proton Steam.

## Установка

```sh
git clone https://github.com/hexquant/protonhax.git
cd protonhax
cargo build --release
```

## Использование

```sh
./target/release/protonhax init <cmd>
```

```sh
./target/release/protonhax run <cmd>
```
